### 디랙토리 구조

```bash
.
├── components (일반 켬포넌트)
│   ├── auth
│   ├── common
│   └── feature...
├── lib
│   ├── api (API 모듈)
│   ├── hooks (Custom Hooks)
│   └── styles (스타일시트)
├── pages (페이지 컴포넌트)
└── store (리덕스 관련 코드)
    ├── app
    └── feature...
```

### 기술스택

- **코어**: React, TypeScript

- **상태관리**: Redux with RTK, Redux Thunk

- **스타일링**: Styled-components

- **패키지 매니저**: Yarn Berry
